🎬 MoodFlix

MoodFlix is a mood-based movie discovery web application where users can find movies according to their mood and manage a personal watchlist.

---

🚀 Features

- 🔐 User Signup & Login (JWT Authentication)
- 🎭 Mood based movie recommendations
- 🔎 Movie search
- ❤️ Personal watchlist
- 🌙 Dark / Light mode toggle
- 🎥 Movie posters
- 👤 Profile page

Install dependencies

Start backend server

Open frontend
